# Dashpaste

![screenshot](https://web.archive.org/web/20160322231511im_/http://images.apple.com/downloads/dashboard/developer/images/dashpaste_20080707123824.jpg)

## About Dashpaste

Extremely useful for exchanging text or code via instant messaging or email. Often it is impossible to paste the text directly because its too long or brakes the formatting of a page. Dashpaste instantly publishes content of user’s pasteboard to popular code sharing sites Rafb.net, Pastebin.ca, Nopaste.info. Also you can set appropriate syntax highlighting (C++, PHP, Ruby, SQL and others). The widget automatically copies the link to the pasteboard for sharing. No more there is a need to open web browser when exchanging code.

## How to run (tested on Tahoe)

Install the widget in [Widget Porting Toolkit](https://github.com/nikolan123/WidgetPortingToolkit). Enable "Proxy XMLHttpRequest" and "Don't ask when running system commands".

## Patch Information

The original pastebin services were replaced with upaste.de and paste.rs.
